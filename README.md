# StrayPups Big Munny v3.0

A compact Class II Bingo/slot hybrid game packaged for easy deployment.

## Contents
- `index.html` — main game page
- `css/styles.css` — extracted game styling
- `js/game.js` — extracted game logic
- `assets/` — image and audio assets used by the game
- `PHASE_PLAN.md` — version notes and design details

## Usage
1. Open `index.html` in a browser.
2. The game should load its assets from the `assets/` folder.

## Notes
- The inline CSS and JS were moved into external files for cleaner maintenance.
- Audio/image files remain in `assets/`.
- The project is delivered as a complete static web package.
